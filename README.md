## 기술스택

- Firebase Firestore
  - 키워드 정보 및 채팅방 정보 저장
- Jetpack AAC
  - Navigation을 사용한 Fragment간 이동 제어
- Hilt
  - Factory 생성 없이 data 모듈까지 편리하게 의존성 주입
- Kakao Local API
  - 장소 검색
- Google Map SDK
  - 지도 표시, 마커 클러스터링, 도로명주소 변환
- Google Auth API
  - 구글 로그인
- Stream Chat SDK & API
  - 복잡한 백엔드 구현과정 없이 채팅 기능을 제공
- Asynchronous(Coroutine, Flow)
